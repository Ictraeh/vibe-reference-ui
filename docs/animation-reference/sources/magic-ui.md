# Magic UI

- **Docs hub:** [magicui.design/docs/components](https://magicui.design/docs/components)
- **URL pattern:** `https://magicui.design/docs/components/{slug}` (kebab-case slug)
- **Install (typical):** `npx shadcn@latest add @magicui/{slug}` — see each doc page.

Use pages for **props** (duration, colors, pause-on-hover) and **ideas**, not as a copy-paste source unless your license workflow allows it.

| Scroll-based text | `interaction` |
