# Kokonut UI

- **Installation / registry:** [kokonutui.com/docs](https://kokonutui.com/docs) (registry: `https://kokonutui.com/r/{name}.json`)
- **CLI pattern:** `npx shadcn@latest add @kokonutui/{component-name}` (see each doc page)

## Doc URL pattern

`https://kokonutui.com/docs/{category}/{slug}`

Categories commonly include: `backgrounds`, `cards`, `navigation`, `inputs`, `texts`, `buttons`, `ai`, …

## Verified examples (copy URL into prompts)

## Discovering more

Use the site nav or search; if a slug is unknown, open the category index on [kokonutui.com/docs](https://kokonutui.com/docs) and follow the sidebar. Names in marketing (“Typing Text”) may use a different slug than the English title—confirm on the live doc.
