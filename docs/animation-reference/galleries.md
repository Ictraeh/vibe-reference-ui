## Workflow for vibe coding

1. User picks a vibe (“Magic UI–style border beam”, “React Bits–style text scramble”).
2. Open the matching **source index** under `sources/` for doc URLs and slug patterns.
3. Map to a **pattern id** in `patterns.json` (or add a new one).
4. Implement with **Motion/CSS first**; only escalate per pattern notes.
